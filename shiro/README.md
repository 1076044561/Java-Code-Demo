SpringBoot-Shiro 
这是一个由SpringBoot搭建的一个学习Shiro的项目。

# 使用技术

* 编译环境：IDEA
* 项目构建工具：Maven
* 框架：Spring + Spring MVC + SpringBoot
* 安全框架：Shiro
* 持久层：Mybatis 
* 数据库：MySql,Redis
* 前端：Html , Jquery ,CSS
